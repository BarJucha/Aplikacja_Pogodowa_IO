## Opis

Mikroprojekt webowej aplikacji pogodowej, umożliwiającej użytkownikowi sprawdzanie aktualnych warunków pogodowych dla różnych lokalizacji. Zintegrowana z bazą danych stworzonej w XAMPP'ie.
